## 문자열 다루기 기본
문자열 s의 길이가 4 혹은 6이고, 숫자로만 구성돼있는지 확인해주는 함수, solution을 완성하세요. 예를 들어 s가 a234이면 False를 리턴하고 1234라면 True를 리턴하면 됩니다.

[ 해결 순서 ] </br>
1. 먼저 길이가 4인지 6인지 

```java
class Solution {
    public boolean solution(String s) {
        boolean answer = true;
        if(s.length()==4){
            for(int i=0;i<4;i++){
                if(s.charAt(i)<'0'||s.charAt(i)>'9')
                    answer=false;
            }
        }
        else if(s.length()==6){
            for(int i=0;i<6;i++){
                if(s.charAt(i)<'0'||s.charAt(i)>'9')
                    answer=false;
            }
        }
        else
            answer=false;
        return answer;
    }
}
```
